Bad Request

HTTP 400
Error Code: badRequest

What happened?

The server could not process your request because it contains invalid syntax, missing required parameters, or violates the request format requirements.

What can you do?

  • Check that all required parameters are included in your request
  • Verify that your request body is properly formatted (e.g., valid JSON)
  • Ensure that the Content-Type header matches the request body format
  • Review the API documentation to confirm the correct request structure
  • Check for any validation errors in the request data
