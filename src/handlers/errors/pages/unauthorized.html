Unauthorized

HTTP 401
Error Code: unauthorized

What happened?

Your request could not be authenticated. This means the server could not verify your identity or the authentication credentials provided were invalid or missing.

What can you do?

  • Ensure you have provided valid authentication credentials
  • Check that your access token or API key is correct and not expired
  • If using Basic authentication, verify your username and password are correct
  • If using Bearer tokens, ensure the token is valid and properly formatted
  • You may need to log in again to obtain a new access token
